My Credit Card Fraud Detection System built for my AI Practical Exam submission. This Project aims to find out the fraudulent transactions that may occur anytime. Please feel free to offer any suggestions.
## Dataset
The dataset is not included due to GitHub size limits.

Download from:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the file at:
data/creditcard.csv
To run this application, please ensure you have these libraries installed:
streamlit, numpy, seaborn, matplotlib, pandas, os, joblib, requests
To install them all, run the command:
pip install requirements.txt
